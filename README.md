## 🧰 Simple troubleshooting

### The app does not open
- Make sure you opened the main app file
- Check that the files are fully extracted
- Try running the file again as admin
- Restart Windows and try once more

### Windows blocks the file
- Right-click the file and choose Properties
- Look for an Unblock option if Windows shows one
- Select Apply, then open the file again

### The screen looks cut off
- Set your display scale to 100% or 125%
- Maximize the window
- Change your screen resolution if needed

### The app feels slow
- Close other apps
- Restart the PC
- Make sure you have enough free memory
- Use a local drive with free disk space

### You cannot find the download
- Open your Downloads folder
- Sort files by date
- Look for a ZIP, EXE, or MSI file with the app name
- Use the browser download history if needed
